Indent Purple Rainbow — это расширение для Visual Studio Code, которое добавляет красивую анимированную фиолетовую радугу к отступам в коде.
Плагин не просто раскрашивает уровни отступов — он делает плавную горизонтальную «бегущую» радугу, где каждый блок из 4 пробелов получает свой оттенок, который со временем смещается, создавая живой градиент.

Подходит для тех, кто хочет улучшить визуальное восприятие структуры кода и сделать редактор более стильным и удобным.
